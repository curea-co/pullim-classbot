## 1. 풀림 플랫폼 개요

### 1.1 한 줄 정의

> **AI가 만들고, 전문가가 검증하고, 데이터가 증명하는 — 교육 콘텐츠의 생산·유통·소비 통합 플랫폼**

### 1.2 3축 구조

```
┌──────────────┐   ┌──────────────┐   ┌──────────────┐
│ 풀림 스튜디오  │──▶│  풀림 스토어   │──▶│  풀림 스터디   │
│  (생산)       │   │  (유통)       │   │  (소비)       │
│              │   │              │   │              │
│ • AI 문제생성 │   │ • 마켓플레이스 │   │ • 적응형 학습  │
│ • 자료 제작   │   │ • 커머스      │   │ • AI 튜터     │
│ • 검증 시스템 │   │ • 크리에이터   │   │ • 진단·분석   │
│ • OCR 변환   │   │ • 구독 서비스  │   │ • 앱 생태계   │
└──────┬───────┘   └──────┬───────┘   └──────┬───────┘
       │                  │                  │
┌──────┴──────────────────┴──────────────────┴──────┐
│          공유 인프라 (CUREA DEEP · 분류DB · 크레딧)    │
├───────────────────────────────────────────────────┤
│      데이터 플라이휠 (학습→개선→품질↑→유입↑→학습)      │
└───────────────────────────────────────────────────┘
```

### 1.3 CUREA DEEP 엔진의 3축 역할

| 축 | CUREA DEEP의 역할 |
|---|-----------------|
| 스튜디오 | 콘텐츠 생성 엔진 (문제 생성, 해설, 분류) |
| 스토어 | 품질 평가 엔진 (인증, 스코어링) |
| 스터디 | 적응형 출제 엔진 (IRT, 실시간 생성, 튜터) |

### 1.4 데이터 플라이휠

```
스터디 (학습 데이터: 정답률, 풀이 패턴, 취약점, AI 튜터 피드백)
    ↓
CUREA DEEP (AI 모델 개선: 더 정확한 문제, 더 나은 해설, 더 정밀한 IRT)
    ↓
스튜디오 (콘텐츠 품질↑: 고품질 문제, 정확한 메타데이터, 크리에이터 생산성↑)
    ↓
스토어 (콘텐츠 풍부↑: 더 많은 상품, 실증 효과 리뷰, 크리에이터 수익↑)
    ↓
스터디 (사용자↑: 더 나은 적응형, 더 나은 코칭, 더 많은 학습자 유입)
    ↓ 루프 반복
```

## 2. 시장 환경

### 2.1 시장 규모

| 시장 | 2025 규모 | 2030 전망 | CAGR |
|------|----------|----------|------|
| 글로벌 AI in Education | $7.05B | $32.27B | 35.5% |
| 글로벌 EdTech 전체 | $340B | $550B | 10.1% |
| 한국 EdTech 전체 | $6.2B | $10.4B | 9.0% |
| 한국 사교육 시장 | ₩26조+ | ₩30조+ | ~3% |
| 한국 인강 시장 | ₩3조+ | ₩4조+ | ~5% |

### 2.2 TAM/SAM/SOM

| 구분 | 시장 | 규모 |
|------|------|------|
| **TAM** | 한국 EdTech 전체 | $6.2B |
| **SAM** | 한국 K-12 AI 학습 + 교사/콘텐츠 | $1.5B |
| **SOM** | AI 문제 생성 + 교재 유통 + 온라인 학습 | $150M |

### 2.3 타이밍이 지금인 이유

- **정책**: 교육부 AI 디지털교과서 단계 도입(2025~), 2026년 693억 원 디지털 교실 인프라 투자
- **기술 성숙**: 하버드 RCT에서 AI 튜터링 학습 효과 2배 실증(2025), 교사 AI 활용률 1년 만에 25%→53%
- **경쟁 공백**: "AI 콘텐츠 생산 + 마켓플레이스 + 학습 플랫폼" 통합 플레이어가 한국에 아직 없음
- **서술형 수능 전환**: 서술형 문제 생성·채점 시장이 블루오션

### 2.4 경쟁사 매핑

```
           콘텐츠 직접 생산
                ↑
  풀림 스튜디오   │   비상/금성/천재 (출판사)
  (AI 생성)      │   (인력 기반 제작)
                │
플랫폼 통합 ─────┼──────── 단일 기능
                │
  풀림 통합       │   콴다/뤼이드
  (생산+유통+소비)│   (풀이/추천 단일 기능)
                │
           콘텐츠 중개/유통
                ↑
  풀림 스토어     │   크몽/클래스101
  (교육 특화)     │   (범용 마켓플레이스)
```
